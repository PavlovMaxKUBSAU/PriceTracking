Приложение выполнено на ASP.NET с использованием swagger UI.
Луковая архитектура (модель, сервисный слой, контроллеры, UI).

Алгоритм работы приложения:
1. Запуск службы.
2. Выполнение асинхронного метода, который проверяет изменения цен на квартиры. В случае положительного результата отправляет информацию на почту того, кто следит за квартирой.
3. Построение графического интерфейса swagger UI.

Выполняемые операции:
Проверка на изменения цен выполняет каждый час.
При регистрации (метод POST) цена за квартиру первый раз высылается мнгновенно на почту.
Метод GET получает данные о квартирах для конкретного email.

Не удалось реализовать:
Парсить цену на квартиру через html оказалось трудоемкой задачей, т.к. она загружается после отработки js. В методе "GetCurrentPriceAsync" я описал разные способы получения цены, но ни один из них не работает. Или цены нет, или ошибка.
API для получения цены найти не удалось.
